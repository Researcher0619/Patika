# Matrix Transposition

This Java program takes user input for the number of rows and columns, creates a matrix, and prints its transpose. It demonstrates the creation and manipulation of matrices.

## Features:

User-input matrix dimensions.
Display of the original matrix.
Calculation and display of the transpose matrix.

### How to Use:

1-Run the program.

2-Enter the number of rows and columns.

3-View the original and transposed matrices.

Enjoy exploring matrices in Java! :)
